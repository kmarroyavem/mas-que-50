# Más que 50: Tu historia continúa

Versión final organizada para GitHub Pages.

## Archivos incluidos

- `index.html`: página principal.
- `styles.css`: diseño responsive, accesibilidad y Story Time mejorado.
- `script.js`: menú móvil, accesibilidad y animaciones.
- `img/imgen.png`: imagen principal de la mujer para el inicio.
- `img/emprender.mp4`: este archivo debes subirlo tú en GitHub con ese nombre exacto.

## Importante sobre el video de emprendimiento

La página ya busca el video en esta ruta:

```text
img/emprender.mp4
```

Por eso debes subir tu video con el nombre exacto `emprender.mp4` dentro de la carpeta `img`.

## Encuestas de Google Forms

La encuesta inicial y la encuesta final ya están integradas dentro de la página mediante iframe y además tienen botón alterno para abrirlas en otra pestaña.

## Publicación en GitHub

1. Entra a tu repositorio.
2. Reemplaza `index.html`, `styles.css` y `script.js`.
3. Verifica que exista la carpeta `img`.
4. Sube `imgen.png`.
5. Sube tu video como `emprender.mp4`.
6. Haz clic en `Commit changes`.

## Recomendación

Si el video de emprendimiento pesa mucho, comprímelo antes de subirlo para que cargue rápido en celular.
