# Más que 50: Tu historia continúa

Página web estática lista para publicar en GitHub Pages.

## Archivos
- index.html
- styles.css
- script.js

## Publicación rápida
1. Crear repositorio en GitHub.
2. Subir estos tres archivos.
3. Entrar a Settings > Pages.
4. En Branch seleccionar main y /root.
5. Guardar.
6. Copiar la URL pública.
